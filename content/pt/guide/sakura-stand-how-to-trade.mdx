<Section icon={Map} title="Entendendo o Sistema de Trocas do Sakura Stand">

O Sakura Stand possui uma economia de trocas altamente ativa, mas sua execução é muito diferente da maioria das experiências do Roblox. Em vez de clicar em um jogador ou acessar um menu dedicado de central de trocas com um único botão, você deve utilizar comandos de chat específicos para iniciar, aceitar e finalizar transações. Essa escolha de design evita trocas acidentais, mas pode ser confusa para novos jogadores que não estão familiarizados com mecânicas de linha de comando.

O sistema de trocas foi construído para facilitar trocas equivalentes de Stands, Specs, skins e itens raros de progressão. Compreender a estrutura subjacente, os tempos de recarga e as opções de interface é essencial para evitar a perda de itens de alto valor em acordos injustos.

| Métrica | Detalhes |
| :--- | :--- |
| Comando Principal | !trade [Nome de Usuário do Roblox] |
| Comando de Confirmação | !confirm (usado se o botão da interface bugar) |
| Comando de Recusa | !decline ou recusar via interface |
| Maestria Mínima | Maestria 1 (requer alcançar o Nível 50) |
| Tempo de Recarga do Sistema | 1 minuto entre tentativas de transação |
| Janela de Cancelamento | 10 segundos para recusar após aceitar |

<Callout type="tip" title="Dica de Mestre">
Sempre planeje suas transações com antecedência. Como o sistema impõe um tempo de recarga estrito de um minuto entre as tentativas de troca, pequenos erros de digitação ou cancelamentos de última hora forçarão você a esperar antes de tentar novamente.
</Callout>

</Section>

<Section icon={Target} title="Como Trocar no Sakura Stand: Passo a Passo">

Executar uma troca com sucesso exige precisão, pois o sistema depende muito da digitação correta dos detalhes no menu de chat global. Se você cometer um erro no nome de um jogador, o comando falhará. Siga este processo estruturado para iniciar e concluir suas trocas com segurança.

<Steps>
  <Step num="1" title="Encontre um Parceiro de Troca">
    Encontre um jogador para negociar no seu servidor atual ou através dos canais de troca na comunidade oficial do Discord de Sakura Stand. Concorde com os itens e valores antes de iniciar a sequência de comandos.
  </Step>
  <Step num="2" title="Obtenha o Nome de Usuário do Roblox">
    Passe o mouse sobre o personagem do jogador no jogo ou verifique a lista de jogadores no canto superior direito da tela. Você deve usar o **Nome de Usuário do Roblox** exato dele, não o nome de exibição (display name) personalizado ou tag de jogo.
  </Step>
  <Step num="3" title="Insira o Comando de Troca">
    Abra a janela de chat global e digite `!trade [NomeDeUsuario]` (substituindo [NomeDeUsuario] pelo nome de usuário real do Roblox do seu parceiro). Por exemplo, se estiver trocando com um jogador chamado John, digite `!trade John`.
  </Step>
  <Step num="4" title="Peça para seu Parceiro Retribuir">
    O outro jogador deve digitar exatamente o mesmo comando de volta para você, referenciando o seu nome de usuário do Roblox (ex: `!trade SeuNomeDeUsuario`). Assim que ambos os comandos forem registrados pelo servidor, a janela de interface de troca se abrirá automaticamente em ambas as telas.
  </Step>
  <Step num="5" title="Adicione os Itens e Confirme">
    Arraste ou clique em seus itens para colocá-los nos slots de troca. Verifique a oferta do outro jogador em relação aos valores da comunidade. Clique no botão verde "Accept" (Aceitar). Se o botão da interface falhar em registrar devido a um bug comum de exibição em dispositivos móveis, digite `!confirm` diretamente no chat para contorná-lo.
  </Step>
</Steps>

<Callout type="warning" title="Aviso Crítico">
Golpistas costumam usar nomes de exibição quase idênticos aos de negociadores de alto valor, enquanto usam nomes de usuário do Roblox completamente diferentes. Sempre verifique o nome de usuário na janela de troca antes de clicar em confirmar.
</Callout>

<Section icon={Shield} title="Requisitos de Troca e Restrições de Maestria">

Para proteger a economia do jogo contra bots, explorações e abusos de transferência direta por meio de contas secundárias, os desenvolvedores implementaram barreiras rígidas de progressão no sistema de trocas. Você não pode simplesmente criar uma nova conta do Roblox e trocar imediatamente itens raros para o seu perfil principal.

O requisito básico para trocar qualquer item ou moeda é a **Maestria 1**. Para obter Maestria, você deve atingir o nível máximo de 50, abrir o menu de configurações e selecionar "Upgrade Mastery" (Melhorar Maestria). Isso redefine o nível do seu personagem de volta ao 1, mas aumenta permanentemente o seu nível de Maestria em 1.

Além disso, Specs e Stands específicos de fim de jogo possuem requisitos de Maestria muito mais altos para evitar que contas de nível baixo possuam habilidades extremamente fortes. Tanto quem envia quanto quem recebe deve atender a esses limites específicos de Maestria para concluir a troca.

| Nome da Spec / Stand | Maestria Requerida para Trocar | Classe da Categoria de Troca |
| :--- | :--- | :--- |
| Cid Kagenou (Shadow) | Maestria 12 | Spec de Alto Nível |
| Sakuya | Maestria 12 | Spec de Alto Nível |
| Binah (The Arbiter) | Maestria 1 | Âncora de Alto Valor |
| Sandevistan | Maestria 1 | Spec de Nível Médio |
| Ryomen Sukuna | Maestria 1 | Spec de Nível Baixo |

<Callout type="info" title="Retenção de Maestria">
Se você realizar um "Breakthrough" (que redefine seu nível e pontos de maestria assim que você acumula 15 maestrias), seus privilégios de troca desbloqueados e itens de inventário permanecem completamente intactos.
</Callout>

</Section>

<Section icon={Trophy} title="Lista de Valores de Troca do Sakura Stand">

As trocas no Sakura Stand são dominadas por um sistema de valores em pontos criado pela comunidade que varia de **0,5 a 5,0**. O valor de um item é determinado por sua força geral de combate, raridade visual da skin, taxas de drop de banners e limites de circulação atuais.

Ao negociar acordos, os jogadores buscam trocas equivalentes com base nesses valores numéricos. For exemplo, um item com valor definido de 4 geralmente vale dois itens avaliados em 2 cada. No entanto, itens limitados de alta demanda costumam exigir um valor extra, forçando os compradores a pagar a mais para garanti-los.
