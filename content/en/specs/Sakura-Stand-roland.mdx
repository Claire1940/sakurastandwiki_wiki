Once you have collected all nine weapons, you must unlock the Librarian's laptop [1]. The Librarian near the Cafe will present you with a randomized sequence of six colors [1]. To solve this puzzle, you must visit six specific locations across the map, find the hidden numbers associated with each location, and translate your color sequence into a six-digit numeric code [1].

| Color | Location | Clue Location |
| :--- | :--- | :--- |
| **White** | Hueco Mundo | Near the central pillar [1] |
| **Red** | Alleyway / Cafe | Behind the main counter [1] |
| **Green** | Ghetto | On the side of the brick building [1] |
| **Orange** | Library | Inside the main reading room [1] |
| **Blue** | Hall of Heroes | Behind the central statue [1] |
| **Purple** | Coliseum | Near the entrance arches [1] |

| Color Sequence | Location Number | Final Code Output |
| :--- | :--- | :--- |
| Red, Green, Orange, White, Blue, Purple | Red=6, Green=4, Orange=5, White=1, Blue=3, Purple=2 | **645132** [1] |
| White, Blue, Purple, Red, Green, Orange | White=1, Blue=3, Purple=2, Red=6, Green=4, Orange=5 | **132645** [1] |
| Orange, Red, White, Purple, Blue, Green | Orange=5, Red=6, White=1, Purple=2, Blue=3, Green=4 | **561234** [1] |

Keep in mind that the color sequence is randomized for every player, and the numbers assigned to each location can also vary [1]. You must physically visit each location listed in the mapping table and look for the glowing number painted on the wall or hidden behind objects [1]. Once you have documented all six numbers, arrange them in the exact order of the color sequence given to you by the Librarian [1]. Inputting the correct six-digit sequence into the laptop will reward you with the Library Invitation [1].

<Callout type="success" title="Puzzle Solved">
