  <Step num="2" title="Physical Conditioning">

  </Step>

<Section icon={Sparkles} title="Skill Set and Combat Mechanics">

The combat style of **Sakura Stand isagi** is divided into two states: Base Form and Flow Mode. In the Base Form, Isagi uses tactical dribbling and "Direct Shots" to keep enemies at bay. Once the Flow meter is filled, he unlocks "Meta Vision," which allows for near-instant teleportation and devastating cinematic finishers.

### Base Form Move List

| Key | Move Name | Description | Damage Type |
| :--- | :--- | :--- | :--- |
| **M1** | Standard Combo | A 4-hit kick combo using the ball | Melee |
| **E** | Naru-haya Footwork | A high-speed dash that bypasses blocks | Mobility |
| **R** | Direct Shot | A powerful ranged kick with high knockback | Projectile |
| **T** | Tactical Dribble | A short-range AOE that stuns nearby foes | Crowd Control |
| **Y** | Egoist Counter | A defensive stance that punishes attackers | Counter |

### Flow Mode Enhancements

When you activate Flow Mode (typically via the G key once the meter is full), Isagi's eyes glow with the iconic Blue Lock aura. In this state, his cooldowns are significantly reduced, and he gains access to the "Meta Vision" Q-skill.

| Skill | Enhancement | Effect |
| :--- | :--- | :--- |
| **Q (Meta Vision)** | Tactical Insight | Highlights enemy hitboxes and increases speed |
| **H (Two-Gun Volley)** | Ultimate Finisher | A massive cinematic combo dealing 40%+ HP |
| **Z (Blue Lock)** | Area Control | Creates a zone where enemies are slowed |

<Callout type="success" title="Combat Mastery">
The key to winning with Isagi is the "Direct Shot" (R). It has a surprisingly long range and can be used to poke enemies from safety before closing in with the Naru-haya dash (E) for a combo.
</Callout>
